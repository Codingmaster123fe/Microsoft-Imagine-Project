# EduConnect - Imagine Cup Submission

## Overview
EduConnect is an innovative AI-powered learning platform designed to solve the global problem of education inequality. The platform connects students in underserved communities with mentors, AI tutoring, and curated educational resources.

## Problem We're Solving
- **258 million** children are out of school globally
- **Limited access** to quality mentorship in developing regions
- **Resource constraints** prevent personalized learning experiences
- **Digital divide** creates educational inequality

## Our Solution
EduConnect democratizes education through:
1. **AI-Powered Tutoring** - 24/7 personalized learning assistance
2. **Global Mentor Network** - Experienced mentors from around the world
3. **Curated Resources** - Free, high-quality educational materials
4. **Smart Recommendations** - AI that adapts to each student's needs
5. **Progress Analytics** - Data-driven insights into learning patterns

## Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **AI:** OpenAI GPT API
- **Deployment:** Docker + Azure/AWS

## Getting Started

### Quick Start with Docker
```bash
docker-compose up --build
# Backend: http://localhost:5000
# Frontend: http://localhost:3000
```

### Manual Setup
```bash
# Backend
cd backend && npm install && npm start

# Frontend (new terminal)
cd frontend && npm install && npm start
```
